## Requisitos

1. python3+
2. memory_profiler

## Execução do programa

Caso deseje isolar o seu ambiente:

```
python -m venv venv
```

Ative o ambiente virtual, por exemplo no powershell:

```
.\venv\Scripts\Activate.ps1
```

Instale a depedência:

```
pip install memory_profiler
```

Execute o programa:

```
python .\main.py
```
